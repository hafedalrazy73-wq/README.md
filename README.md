# 🍕 Pizza Order & Invoice Management System

![C#](https://img.shields.io/badge/Language-C%23-blue.svg?style=for-the-badge&logo=csharp)
![.NET](https://img.shields.io/badge/Framework-.NET%20%2F%20Windows%20Forms-purple.svg?style=for-the-badge&logo=dotnet)
![Environment](https://img.shields.io/badge/Environment-Visual%20Studio%202026-orange.svg?style=for-the-badge&logo=visualstudio)

A desktop application designed and developed to manage pizza ordering workflows and dynamic, real-time invoice creation. Built using C# and Windows Forms with clean Object-Oriented Programming (OOP) principles.

---

## 🚀 Core Features

- **Dynamic Order Composition:** Fully interactive interface managing different pizza sizes, crust selection, multi-selection toppings, and quantity.
- **Real-time Pricing Engine:** Instant calculation of totals, subtotals, discounts, and taxes dynamically in the background based on system logic.
- **Advanced Invoice Grid:** An integrated data grid view acting as a formatted receipt to summarize all order details before final submission.
- **Input Sanitization:** Strict validation rules implemented across input controls to prevent incorrect data or application crashes.

---

## 🛠️ Project Architecture

The repository contains clean, production-ready source files and structure:
- **OrderPizza.cs & OrderPizza.Designer.cs:** Core logic and automated visual layout components handling event-driven operations.
- **OrderPizza.resx:** XML-based resource file securely embedding visual layouts and graphical assets.
- **My Pizza Project.csproj & My Pizza Project.slnx:** Project configurations optimized for compilation inside Visual Studio 2026.

---

## ⚙️ Installation & Run

Follow these standard steps to pull and execute the project locally on your system:

1. Clone the project repository using Git:
   git clone [https://github.com/hafedalrazy73-wq/Pizza-Order-System.git](https://github.com/hafedalrazy73-wq/Pizza-Order-System.git)

2. Navigate into the project directory on your machine.

3. Open "My Pizza Project.slnx" to load the solution environment in Visual Studio 2026.

4. Press F5 or click "Start" to run the compilation and initiate the software.
